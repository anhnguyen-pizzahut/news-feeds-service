# Newsfeeds Service
### API (NodeJs, SQLite)
Move to **api** folder: `cd api`.

Install packages: `npm i`.

Dev mode: `npm run dev`

To run: `npm start`


### UI (ReactJS)
Move to **ui** folder: `cd api`.

Install packages: `npm i`.

Dev mode: `npm run ssr`

To run: `npm start`

Access `localhost:3000`
